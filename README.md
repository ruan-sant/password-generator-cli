# 🔐 Password Generator CLI

Um gerador de senhas simples via linha de comando feito em Node.js.

## 🚀 Funcionalidades

* Gera senhas aleatórias
* Permite definir o tamanho da senha
* Fácil de usar no terminal

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/ruan-sant/password-generator-cli.git
cd password-generator-cli
```

Instale as dependências:

```bash
npm install
```

## ▶️ Como usar

Gerar uma senha com tamanho padrão (8 caracteres):

```bash
npm start
```

Gerar uma senha com tamanho personalizado:

```bash
npm start 12
```

## 🧪 Testes

Projeto agora possui testes unitários com Jest.
